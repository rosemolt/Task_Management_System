<!-- Task Management System -->
<!-- Project Overview -->

The Task Management System is a full-stack web application developed as part of an internship sample task.
It allows users to log in and manage their tasks efficiently with full CRUD (Create, Read, Update, Delete) functionality.

This project demonstrates practical understanding of Angular, Node.js (Express), and MySQL, along with clean project structure, API design, and basic authentication.

<!-- Backend Setup (Node.js + Express) -->

Install dependencies:

npm install

Start backend server:

npm start

Backend runs at:

http://localhost:3000

<!-- Frontend Setup (Angular) -->

Install dependencies:

npm install

Run Angular app:

ng serve

Frontend runs at:

http://localhost:4200

<!-- API Endpoints -->
Method	Endpoint	Description
POST	/login	    User login
GET	    /tasks	    Get tasks
POST	/tasks	    Create task
PUT	    /tasks/:id	Update task
DELETE	/tasks/:id	Delete task

<!-- API Testing -->

Used Postman to test APIs.
